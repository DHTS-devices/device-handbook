# Oculus Rift S

---

# 1. Overview

- Device name:
  Oculus Rift S

- Device type:
  PC VR Headset

- Category:
  XR / VR

- Manufacturer:
  Oculus (later Meta)

- Overview:
  The Oculus Rift S is a PC-powered virtual reality headset designed for:
  - VR gaming
  - simulation
  - VR development
  - education
  - research
  - immersive visualization

  The Rift S is the successor to the original Oculus Rift (CV1) and introduced:
  - inside-out tracking
  - simplified setup
  - improved display clarity
  - reduced hardware requirements

  Unlike the original Rift, the Rift S does NOT require external sensors.

  Instead, it uses:
  ```text
  onboard cameras
  ```
  for positional tracking.

  The Rift S became popular because it offered:
  ```text
  EASY PC VR
  ```
  without requiring:
  - base stations
  - external tracking cameras
  - complex room setup

  Compared with modern standalone headsets like Quest, the Rift S is:
  ```text
  PC ONLY
  ```

  meaning:
  ```text
  gaming PC
  +
  Oculus software
  ```
  are required.

# 8. STEP-BY-STEP First Setup

---

## STEP 1 — Install Oculus Software

Download:

```text
Meta Quest / Oculus PC App
```

Install on Windows PC.

---

## STEP 2 — Connect Headset

Connect:

```text
DisplayPort → GPU
USB → PC
```

Important:
Always connect DisplayPort to:
```text
dedicated GPU
```

not motherboard ports.

---

## STEP 3 — Open Oculus App

Launch Oculus software.

If successful:
```text
Headset Connected
```

appears.

---

## STEP 4 — Pair Controllers

Insert batteries into controllers.

Usually:
```text
automatic pairing
```

If not:
1. Open Devices
2. Select controller pairing
3. Follow instructions

---

## STEP 5 — Update Firmware

The software may update:
- headset
- controllers

Allow updates to finish.

