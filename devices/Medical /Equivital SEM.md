# 8. IMPORTANT: Belt Fit Matters

This is VERY important.

Equivital works best when:

```text
belt fits tightly
```

Why?

Because ECG electrodes require:

```text
GOOD CHEST CONTACT
```

Loose fit causes:

- ECG noise
- respiratory artifact
- signal dropout

For research:

Always record:

```text
belt size
tightness
participant fit
```

before study.
