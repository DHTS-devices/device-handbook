The GT9X Link includes:

```text

```

Measures:

```text

acceleration
```

Dynamic range:

```text
±8G
```

This is the:

```text
MAIN SENSOR
```

```text

+
POSITION

```text
±2000°/sec
```

Magnetometer range:

```text
±4800 µT

Useful for:

```text

fall detection
posture
```.
