# 8. IMPORTANT: Sensor Placement Matters

This matters A LOT.

The finger sensor must:

```text
ALIGN CORRECTLY
```

Bad alignment causes:

```text
incorrect readings
signal loss
poor data
```

The sensor LED and detector must remain:

```text
properly aligned
```

throughout the night. Bright external light may interfere with measurements.
